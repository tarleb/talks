# Pandoc

## Der Ultimative Konverter

- Konvertiert zwischen zahlreichen Formaten
- Nicht immer perfekt, aber extrem vielseitig
- Erweiterbar
  + Haskell
  + Python
  + Lua

## Bekannte Eingabeformate

- Markdown
- HTML
- \LaTeX
- Docx (Word)
- ODT (Open Office)
- Emacs Org-Mode

## Nützliche Ausgabeformate

Alle Eingabeformate, plus

- Ebook
- PDF
- Slideshow
  + Reveal.js
  + Beamer
